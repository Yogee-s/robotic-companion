# URex technical report

Self-contained HTML report. Open `index.html` in a browser.

To convert to PDF: open in Chromium, **Ctrl-P** → Paper A4, Default margins, **Background graphics on**, Save as PDF. The print stylesheet hides the sidebar TOC, adds a footer with the GitHub URL and page number, and forces page breaks between chapters.

To serve locally: `python3 -m http.server 8000` in this folder, then open `http://localhost:8000`.
